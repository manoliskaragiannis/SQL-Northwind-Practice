Αυτό το αποθετήριο περιέχει μια σειρά από ασκήσεις SQL πάνω στη βάση δεδομένων NORTHWIND
Σκοπός είναι η εξάσκηση σε πραγματικά σενάρια διαχείρισης δεδομένων ενός συστήματος ERP.



ΤΕΧΝΟΛΟΓΙΕΣ:

SQL Server Management Studio
T-SQL
GitHub


Θεματολογία που καλύφθηκε:

Data Retrieval: Χρήση 'SELECT' , 'WHERE' , 'ORDER BY' , 'TOP'
Table Relations: 'INNER JOIN' , 'LEFT JOIN'
Data Aggregation: 'GROUP BY' , 'COUNT' ,'SUM'
Business Logic: 'CASE'


Μπορείτε να βρείτε όλες τις λύσεις των ασκήσεων στο αρχείο `Northwind_Exercises.sql`. 
Κάθε query είναι σχολιασμένο για την κατανόηση της λειτουργίας του.
